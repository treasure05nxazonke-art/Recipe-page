# Frontend Mentor – Recipe Page Solution

This is my solution to the **Recipe Page challenge** from Frontend Mentor.  
The goal of the project was to build a responsive recipe page that closely matches the provided design.

---

## 📚 What I Learned

While working on this project, I improved my understanding of several CSS concepts.

One of the key things I learned was how to properly align table content using column widths and spacing. I also practiced styling list markers and controlling spacing between bullet points and text.

Another important lesson was making images responsive so they adapt to different screen sizes without breaking the layout.

---

## 🚧 Challenges I Faced

One of the biggest challenges was aligning the nutrition values in the table so that all numbers started at the same position. I experimented with padding, text alignment, and column widths before finding a clean solution.

Another challenge was making sure the layout looked good on both desktop and mobile screens. I solved this by using flexible widths and media queries.

---

## 🔮 Continued Development

In future projects, I would like to:

- Improve my CSS structure and organization
- Practice writing more scalable and reusable styles
- Get better at responsive design without relying too heavily on media queries
